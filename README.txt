1. Create a folder named "gitToGithub" in the local machine
2. Create a file named "README.txt" and add some contents into it
3. open git bash and navigate into the folder "gitToGithub"
4. git init
5. git branch # check if the branch name is same as the branch name in Github, if not rename
6. git branch -M main
7. Go to the Git repository where you want the local machine code to be pushed.
8. Copy the SSH link git@github.com:NaveenSilvester/Allen.git
9. git remote add origin git@github.com:NaveenSilvester/Allen.git
10. git remote -v
11. git pull 
12. git branch --set-upstream-to=origin/main
13. git pull
14. git push 
15. git add .
16. git commit 
17. git push
